# Blue-Print 0.9.0 リリース候補

## 対応範囲

- macOS 14以降、Apple Silicon
- 2025年分の事業所得・不動産所得用青色申告支援
- 弥生デスクトップ／オンライン25列、弥生会計 Next 25／27列仕訳と6列期首残高
- DBスキーマ9、持ち出し形式8

## 主な改善

- 10万仕訳明細・2万証憑の標準性能データと自動ベンチマーク
- 仕訳検索のN+1解消、32MB SQLiteキャッシュ、64MB証憑画像キャッシュ
- 全データ出力・暗号化バックアップ・診断をバックグラウンド化
- `⌘1`〜`⌘5`の主要画面移動、VoiceOverラベル、色以外の状態文言
- エラーを「原因・影響・修正方法」で表示
- 公式ビルドと自己ビルドの識別、署名・公証・チェックサム生成スクリプト

## 既知の制限

- `.xtx`の生成・内部検証は完了しているが、2025年分e-Tax WEB版への実地読込は利用者のe-Tax環境で未確認。
- Developer ID証明書とNotary Service資格情報がこの環境にないため、署名・公証済み公式バイナリは未作成。手順と自動化スクリプトは検証済み。
- Intel Macは正式対象外。macOS 14〜16のうち、この環境ではmacOS 26/Xcode 26.2のみ実機検証。
- OCR候補はオンデバイス生成で、税務判断や自動確定は行わない。
- 自動バックアップはパスフレーズをKeychainへ登録した後に有効になる。v1では初回案内を追加する。

## 税制・e-Tax根拠

最終確認日: 2026-07-21。コード内の`RuleSource`から国税庁「令和7年分 確定申告特集」とe-Tax仕様を参照できる。弥生形式は同日、弥生公式サポートの25／27列・期首残高・税区分仕様を確認した。
