# Blue-Print 1.0.0 公開案内（公開ゲート待ち）

## 対応環境

- macOS 14 Sonoma以降、Apple Silicon
- 正式検証: macOS 26 / Xcode 26.2 / Swift 6.2
- Intel Macは対象外

## 対応年度・形式

- 2025年分の事業所得・不動産所得用青色申告支援
- 給与、株式、配当、その他所得の年度集約
- 弥生デスクトップ／オンライン25列仕訳
- 弥生会計 Next 25／27列仕訳、6列期首残高
- Blue-Print DBスキーマ9、持ち出し形式8

## 配布物

公開時に以下を同じ`v1.0.0`版で提供する。

- Developer ID署名・Apple公証済み`BluePrint-macOS-arm64.zip`
- `BluePrint-macOS-arm64.zip.sha256`
- GitタグとGitHub自動生成ソースアーカイブ
- 本リリースノート
