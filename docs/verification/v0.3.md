## 自動検証

```sh
swift format lint --strict --recursive Sources Tests Package.swift
env CLANG_MODULE_CACHE_PATH=/tmp/blueprint-clang-cache \
  SWIFTPM_MODULECACHE_OVERRIDE=/tmp/blueprint-swiftpm-cache \
  swift test --scratch-path /tmp/blueprint-build
env CLANG_MODULE_CACHE_PATH=/tmp/blueprint-release-clang-cache \
  SWIFTPM_MODULECACHE_OVERRIDE=/tmp/blueprint-release-module-cache \
  swift build -c release --scratch-path /tmp/blueprint-release-build
rg -n 'URLSession|URLRequest|NWConnection|Network\.framework|WebSocket|HTTPURLResponse' \
  Sources Package.swift
```

- Swift Format strict lint: 成功
- 全45テスト: 成功、失敗0

## v0.3 完了条件の対応

| 完了条件 | 検証 |
| --- | --- |
| 同一原本を重複として扱う | SHA-256 一致時に既存IDを返すエラーを確認 |
| OCR修正後も原本が不変 | 修正前後のSHA-256一致と監査イベントを確認 |
| 10%適格／未登録を別集計 | 税率、インボイス区分、控除割合別の集計を確認 |
| CSV不正行を隔離 | 正常1件と不正1件が同一バッチで独立保存されることを確認 |
| 電子原本の検索・表示 | 日付、金額、取引先、電子取引フラグの複合検索を確認 |
| 通常操作で外部通信なし | ソースと依存定義に通信APIがないことを確認 |
| ロック年度の取込禁止 | 証憑／CSV双方が `fiscalYearLocked` になることを確認 |
