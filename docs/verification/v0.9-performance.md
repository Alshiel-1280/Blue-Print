# v0.9 性能検証

Apple Silicon、release構成、2026-07-21に`blueprint-benchmark`を実行した。標準データは5万仕訳（10万明細）と2万証憑、SQLiteファイル約59MB。

| 項目 | 実測 | 基準 | 結果 |
| --- | ---: | ---: | --- |
| DBオープン | 0.001秒 | 3秒 | 合格 |
| 仕訳検索 | 0.105秒 | 1秒 | 合格 |
| 証憑検索 | 0.004秒 | 1秒 | 合格 |
| 主要画面用全仕訳ロード | 1.231秒 | 3秒 | 合格 |
| 月次試算表 | 0.028秒 | 5秒 | 合格 |
| 年次決算書 | 0.886秒 | 10秒 | 合格 |
| CSV 1万行プレビュー | 0.428秒 | 60秒 | 合格 |

ピーク常駐メモリは約350MB。SQLiteページキャッシュを32MB、mmapを64MB、WALを16MBに制限し、証憑画像は最大2048pxへ縮小して24件・64MBで破棄する。仕訳と明細は800件単位の一括読込へ変更し、N+1クエリを除去した。

再現:

```sh
swift run -c release blueprint-benchmark .build/performance
```

機械可読結果は`artifacts/v0.9/performance-report.json`。
