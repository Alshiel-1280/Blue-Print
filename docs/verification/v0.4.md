## v0.4 完了条件の対応

| 完了条件 | 検証 |
| --- | --- |
| 請求確定の冪等性 | 同一IDの発行を2回実行して請求・仕訳・証憑が各1件であることを確認 |
| 一部・分割・合算入金 | 単一請求の複数回消込と、2請求を1仕訳で合算消込することを確認 |
| 手数料・源泉・値引・過不足 | 消込構成要素の貸借一致と残高超過拒否を確認 |
| 外注請求と一部支払 | 未払残高、支払仕訳、源泉既定オフを確認 |
| 訂正・取消・返還 | 元仕訳反転、訂正後仕訳、返還の逆方向仕訳、元請求IDを確認 |
| PDF記載内容 | 登録番号、10%・8%対象額、税額、合計をメタデータと実レンダリングで確認 |
| PDF原本・再発行 | 原本を上書きせず、再発行証憑と理由・連番を保存することを確認 |
