## v0.5 完了条件の対応

| 完了条件 | 検証 |
| --- | --- |
| 固定資産の年度別計算 | 定額・定率・少額・一括償却のゴールデン値と1円残存価額を確認 |
| 取得から売却・除却まで | イベント履歴と取得・売却仕訳の貸借一致、事業割合変更を確認 |
| 家事按分 | 100,000円、家事30%で事業主貸と経費各30,000円を確認 |
| 棚卸・売上原価 | 期首20,000円、仕入100,000円、期末30,000円で売上原価90,000円を確認 |
| 決算テンプレート | 未収・未払・前払・前受の4種が決算仕訳を生成することを確認 |
| P&L/B/S整合 | 当期利益80,000円が貸借対照表の増減と一致し、貸借一致を確認 |
| 年度確定注意 | blocking項目未解決時に確定不可と注意文を返すことを確認 |
| 版情報 | PDFメタデータとCSVにapp・data・tax・form版を記録することを確認 |
