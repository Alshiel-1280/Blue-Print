# ADR-005: 金額・消費税・端数

- 状態: 採用
- 日付: 2026-07-21
- 対象: NFR-COR-001〜002、FR-SET-005、FR-TAX-007

## 判断

- 帳簿金額は `Int64` の整数円を `Money` で包み、`Double` を使用しない。
- 税率は列挙値と基準点で表し、インボイス区分とは独立して保持する。
- 税額計算途中は十進の分子・分母または `Decimal` を使用し、円へ確定する境界で `floor / ceil / nearest` を明示する。
- 丸め単位（明細、請求、伝票）、規則、年度ルール ID・版を計算結果へ記録する。
- オーバーフローは例外として扱い、部分保存しない。
