# ADR-001: OSS ライセンス

- 状態: 採用
- 日付: 2026-07-21
- 対象: FR-OSS-001〜002

## 判断

Blue-Print を Apache License 2.0 で公開する。著作権許諾に加えて明示的な特許許諾を持ち、個人利用、派生版、パッケージ配布、他 OSS との統合を妨げない。

## 比較

- GPL 系は派生物の公開を強制できる一方、macOS 配布物や将来の連携先との互換性判断が増える。
- MIT は単純だが、Apache-2.0 のような明示的特許条項を持たない。
- Apache-2.0 は NOTICE と変更表示が必要だが、採用・再配布と防御的な特許許諾のバランスがよい。

## 帰結

依存ライブラリは Apache-2.0 と両立するライセンスに限定し、リリースごとに第三者ライセンス一覧を生成する。
