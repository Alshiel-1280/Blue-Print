# ADR-009: 証憑・バックアップ暗号化の境界

- 状態: 採用
- 日付: 2026-07-21
- 対象: NFR-SEC-001〜003、FR-BKP-003

## 判断

ローカル正本は macOS の FileVault、ユーザー権限、App Sandbox/Data Protection を前提とし、v1 初期版では証憑ごとの独自暗号化を必須にしない。アプリ独自鍵を導入する場合は Keychain に保存する。

持ち出されるバックアップは必ず認証付き暗号化する。パスフレーズから Argon2id で鍵を導出し、AES-256-GCM でマニフェスト、DB、証憑を一体として保護する。暗号方式とパラメータはバックアップ版に含め、復元前に認証と全ハッシュを検証する。
