# ADR-002: 対応 macOS とツールチェーン

- 状態: 採用
- 日付: 2026-07-21
- 対象: NFR-CMP-001〜002

## 判断

- デプロイ対象は macOS 14 以降とし、リリース時点の最新版と過去2メジャーを CI・実機受入の対象にする。
- Apple Silicon を正式対象とする。Intel は v1 の配布検証で再評価する。
- ソース基準は Swift 6、検証ツールチェーンは Xcode 26.2 / Swift 6.2 とする。
- SwiftUI を UI の中心とし、AppKit は SwiftUI で満たせない macOS 固有機能に限定する。

## 帰結

新しい API は macOS 14 での代替経路を持つ。最新版でしか動かない UI を導入する場合は ADR を更新する。
