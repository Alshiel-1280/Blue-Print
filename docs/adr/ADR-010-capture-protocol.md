# ADR-010: 将来の iPhone 転送と正本

- 状態: 採用
- 日付: 2026-07-21
- 対象: FR-IOS-005〜008、NFR-MNT-004

## 判断

Mac を唯一の正本とし、iPhone は撮影・簡易 OCR・送信待ちだけを行う。共有契約は証憑 UUID、原本 SHA-256、端末 ID、撮影日時、MIME、サイズ、転送状態、プロトコル版を持つ。

転送は端末ペアリング後の相互認証・暗号化チャネルを使用し、`document UUID + content hash` を冪等キーとする。Mac が原本の永続化・ハッシュ検証・受信確認を完了するまで iPhone は削除しない。同期方式は protocol の実装として交換可能にし、iCloud を必須にしない。
