# ADR-006: 監査ログと仕訳訂正

- 状態: 採用
- 日付: 2026-07-21
- 対象: FR-BKP-005〜006、FR-JRN-004〜005、008

## 判断

`AuditEvent` は UUID、日時、操作者種別、対象、操作、変更理由、関連イベントを持つ追記専用レコードとする。SQLite trigger で更新・削除を拒否し、通常 UI に編集機能を持たせない。

転記済み仕訳は物理削除しない。取消は反対仕訳、訂正は元仕訳・取消・新仕訳のリンクで表す。再オープン、年度ロック、マスター無効化も監査イベントへ残す。個人・税務データの全文差分は記録せず、対象 ID と機微でない要約だけを保存する。
