# 新年度対応手順

1. 国税庁とe-Taxの一次資料を保存し、対象年度、URL、確認日を記録する。
2. 既存の`TaxRuleSet`・`FormRuleSet`を変更せず、新しいIDと版で追加する。
3. 前年度との差分を`RuleCatalog.difference`で確認し、影響項目を記録する。
4. 帳票ID、手続ID、項目型、桁数、必須条件を更新する。
5. 一次資料の公開例をゴールデンテストへ追加する。
6. 過年度を含む全テスト、`.xtx`検証、e-Tax WEB実地読込を行う。
7. 根拠URL、確認日、対応年度、追加入力項目と既知の制限をリリースノートへ記載する。

既存年度のルールや期待値を書き換えない。制度変更で再計算が必要な場合は、対象年度と再出力の要否を利用者へ明示する。
